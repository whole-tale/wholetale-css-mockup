Create New Environment

A WholeTale Enviroment is a virtual computer you pair with data to compose a Tale. Environments you and others have created can be selected from the list of Environments when composing a Tale. To create a new Environment, you must provide setup instructions in the form of a Dockerfile that is checked into a Git repository that is web accessible.
