More details

Your published Tale will include everything that has been uploaded to its associated workspace. In addition, the following files will be generated and included in the published Tale. This process allows another user to easily rerun your published analysis using the WholeTale platform.

manifest.json This file holds metadata about the tale, such as script execution order and file structure.
 environment.json This environment file holds the information needed to re-create the tale's base virtual machine.
LICENSE The Tale's license is included in the package. This can be selected from the Tale's metadata view.
README.md The Tale's README is included in the package. The README includes instructions for how to use and run the published analysis encapsulated by your Tale.
metadata.xml The contents of each package are described using the Ecological Metadata Language (EML). To learn more about EML, visit the EML primer.
